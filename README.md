# egg-timer-electron
A minimalist Egg Timer application built with Electron, designed to help users cook eggs perfectly every time. The app offers preset timers for different egg doneness levels, including:
Soft boiled
Medium boiled
Medium-hard boiled
Hard boiled

Each option launches a dedicated countdown timer with a clean UI, animated visuals, and an audio alert when the timer finishes.

#Featuresz;
Simple menu-based navigation
Preset timers for common egg types
Real-time countdown display
Custom-designed app frame, background, and animated GIF
Audio alarm notification when the timer ends
Window controls (minimize, maximize, close) integrated with Electron

#Tech Stack
Electron
HTML, CSS, JavaScript
Electron preload API for window controls
Custom assets (UI frame, background, animation)
